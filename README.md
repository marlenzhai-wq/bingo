# Bingo Bot (aiogram 3.x)

## Орнату

```
pip install -r requirements.txt
```

`config.py` ішінде `BOT_TOKEN`-ды өзгертіңіз немесе `BOT_TOKEN` env-айнымалысын беріңіз:

```
export BOT_TOKEN="123456:ABC-DEF..."
python3 bot.py
```

## Файлдар

- `bot.py` — барлық handler/callback логикасы, бот старты
- `db.py` — SQLite (aiosqlite) дерекқор қабаты: games, players, called_numbers
- `cardgen.py` — карта генерациясы, BINGO тексеру, Pillow арқылы PNG сурет жасау
- `config.py` — токен мен DB жолы

## Қысқаша логика

- `/newgame` — кез келген пайдаланушы админ ретінде ойын ашады, deep-link сілтемесі беріледі.
- `/start game_xxxxx` — ойыншы қосылады, бір рет жеке 5×5 картасы InlineKeyboard түрінде жіберіледі (сурет емес).
- `/next`, `/next2`, `/next3`, `/next4` — тек ойынды ашқан админге кездейсоқ сандар жіберіледі (мысалы B-14). Админ оларды ойыншыларға өзі таратады.
- Ойыншы батырманы басқанда сан шыққан болса ✅ болып белгіленеді, шықпаса ештеңе өзгермейді.
- Жол/баған/диагональ толық белгіленгенде BINGO анықталады, FREE ұяшығы автоматты белгіленген.
- BINGO кезінде жеңіске жеткен ойыншының картасы Pillow арқылы PNG суретке түрленеді (белгіленген — жасыл, жеңіс сызығы — сары/қызыл шектеу) және әкімшіге + барлық ойыншыларға дәлел ретінде жіберіледі.
- `/stop` — ойынды аяқтайды, барлық ойыншыларға хабарлайды.

## Ескертпе

- Бір уақытта бірнеше ойын болуы мүмкін (әр админнің өз `game_id`-сі бар). Бір админде бір мезетте тек бір "активті" ойын болады деп есептелген (`/next*` және `/stop` команда осыдан соңғы активті ойынды табады).
- Дерекқор: SQLite, файл — `bingo.db` (бірінші іске қосуда автоматты жасалады).
